### C# のコーディング規約
- `for` や `if` 等において，処理が1行のみであっても { } を省略しない
- `if` において `!` を極力使用しない<br>例: `if( !a )` -> `if( a != true )` または `if( a == false )`
- `void` のメソッドであっても最後の `return;` を省略しない
- 細かなコードスタイルは Visual Studio のオプション内で設定し，自動書式で整形する
